- ü§ñ **[Integram](https://github.com/horhe-dvlp/Integram)**  
    A FastAPI-based REST API for working with metadata and dynamically constructing SQL queries for PostgreSQL.
    The system provides RESTful endpoints for retrieving information about terms, objects, attributes, and reference tables, as well as creating and executing complex queries with automatic JOIN generation.
    The application uses SQLAlchemy for asynchronous database operations, includes JWT authentication and route-protection middleware, and is deployed via Docker Compose.
    A key feature of the project is the dynamic QueryBuilder, which analyzes relationships between tables and automatically generates optimized SQL queries based on the requested columns.
